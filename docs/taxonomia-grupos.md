# Taxonomía de grupos

Estos son los valores válidos para `grupo_carrusel` en `archivo_x` y en la interfaz de curaduría.

| Grupo | Criterio |
|---|---|
| Amor romántico | Idealizar, ilusionarse, querer amor |
| Desamor y tusa | Duelo, extrañar, dolor emocional |
| Ex y contacto cero | Ex, bloqueo, recaídas, recuerdos |
| Coqueteo y deseo | Flirteo, atracción, tensión |
| Vínculos confusos | Ghosting, casi algo, señales mixtas |
| Sexo y cuerpo | Sexo, intimidad explícita, deseo físico |
| Dinámica de pareja | Reciprocidad, celos, expectativas, responsabilidad afectiva |
| Hombres y género | Hombres, mujeres, roles, quejas, red flags |
| Actitud, autoestima y límites | Orgullo, independencia, dignidad, no rogar |
| Autorretrato y mood | Cómo soy, mood, contradicciones, autosabotaje |
| Salud mental | Ansiedad, terapia, depresión, cansancio vital |
| Universidad y estudio | Parciales, semestre, carrera, estudiar |
| Plata, trabajo y vida adulta | Sueldo, deudas, contratos, trabajo, adultez |
| Bogotá, Colombia y calle | Ciudad, país, clima, cultura local, rolo/parce |
| Fiesta, alcohol y sustancias | Tomar, fumar, vapo, guaro, techno, bares |
| Tecnología, IA y redes | ChatGPT, IA, Instagram, TikTok, apps, reels |
| Política, actualidad y cultura pop | Políticos, farándula, música, memes coyunturales |
| Humor cotidiano y absurdo | Frases random, observaciones raras, chistes sin tema fuerte |
| Familia, amigos y hogar | Mamá, papá, amigos, casa, arrunche, domingo |
| Místico y destino | Tarot, Diosito, Mercurio, luna, señales, espiritualidad |

Reglas rápidas:

- Si menciona explícitamente un ex o contacto cero, usa `Ex y contacto cero`.
- Si el chiste tiene tema claro, usa ese tema antes que `Humor cotidiano y absurdo`.
- `Humor cotidiano y absurdo` es el último recurso.
- Si una frase queda entre amor, deseo y vínculo confuso: deseo físico va en `Coqueteo y deseo`; señales mixtas van en `Vínculos confusos`; ilusión amorosa va en `Amor romántico`.
